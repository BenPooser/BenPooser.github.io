# How to make updates

### `npm start`

To test changes to the app

### `git add .`
### `git commit -m <commit-message>`
### `git push`

To push changes to the app

### `npm run deploy`

To deploy changes on the app to Github Pages

Make sure to add custom domain after each npm run deploy
